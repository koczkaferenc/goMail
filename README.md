# goMail
Levelező rendszer

## A rendszer összetevői

* Exim4 SMTP szerver
* Docvecot IMAP/Pop3 szerver
* ImapSync Levél letöltést végző szerver
* goMail adminisztrációs felület
* mySQL adatbázis szerver
